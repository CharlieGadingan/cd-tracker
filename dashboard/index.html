Git Tutorial

Follow these steps to use Git version control to submit your activity.

Step 1: Open the folder where your project is located.

Step 2: Right-click inside the folder.

Step 3: Click Show more options (if using Windows 11).

Step 4: Click Git Bash Here to open the terminal.

Step 5: Type git add .

This adds all your changed files to the staging area (preparing them to be saved).

Step 6: Type git commit -m "your message"

This saves your changes with a message describing what you did.

Step 7: Type git pull

This gets the latest updates from the repository (important before pushing).

Step 8: Type git push

This uploads your changes to GitHub.

Tip: Want to learn more about using Git and GitHub through Git Bash?

Click the Learn More button to explore a beginner-friendly guide that walks you through essential commands, version control basics, and how to manage your projects efficiently.

Create Classroom Guide

Follow these steps to use Git version control to submit your activity.

Step 1: Click "New Class".

Step 2: Fill up the class details.

Step 3: Click the switch below the "Require Passcode" (Only if creating private room).

Step 4: Click the switch below the "Require Approval" (Only if the creator wants to approve before letting some user get in).

Step 5: Click the "Create Class" to fully build to classroom.

Join Classroom Guide

Follow these steps to use Git version control to submit your activity.

Step 1: Click "Join".

Step 2: Enter class code (Only if your joining private room).

Step 3: Click "Join Class".
